DOSSIER DOCUMENTS
==================

Ce dossier contient les documents PDF de campagne.

Fichiers à ajouter :
--------------------

1. programme.pdf (16 pages)
   - Programme complet de la campagne
   - Propositions détaillées pour Senlis

2. lettre-habitants.pdf
   - Lettre d'engagement aux Senlisiens

3. tract.pdf

   - Version synthétique du programme

Instructions :
--------------
1. Ajoutez vos fichiers PDF dans ce dossier
2. Respectez les noms de fichiers indiqués ci-dessus
3. Les liens dans le site web pointent vers ces fichiers

Note : Les fichiers PDF doivent être optimisés pour le web (compression) pour de meilleures performances.
