DOSSIER IMAGES
==============

Structure des dossiers :
-------------------------

/images
  ├── PSEC.png (Logo - déjà présent)
  ├── qr-code.png (Généré automatiquement par le site)
  ├── /gallery (Photos de campagne)
  ├── /candidates (Photos des candidats de la liste)
  └── /placeholder (Images temporaires)

Instructions pour ajouter des images :
---------------------------------------

1. PHOTO OFFICIELLE DE LA CANDIDATE
   - Ajouter : candidate-officielle.jpg
   - Format recommandé : Portrait vertical, 600x800px minimum
   - Remplacer le placeholder dans index.html (ligne 95)

2. GALERIE DE CAMPAGNE (/gallery)
   - Ajoutez vos photos d'événements
   - Format : JPG ou PNG, optimisées pour le web
   - Taille recommandée : 1200px de largeur maximum
   - Les photos seront affichées dans une grille responsive

3. CANDIDATS (/candidates)
   - Photos des membres de la liste
   - Format : Portraits carrés, 500x500px minimum
   - Nommage : prenom-nom.jpg

Optimisation des images :
--------------------------
- Utilisez des outils comme TinyPNG ou ImageOptim
- Format JPEG pour les photos (qualité 80-85%)
- Format PNG pour les logos et graphiques
- WebP pour une meilleure compression (optionnel)

Note : Les images non optimisées peuvent ralentir le chargement du site.
